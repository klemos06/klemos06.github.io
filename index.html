// 01 — ABOUT

Who I am

I'm Keegan Lemos, an electrical engineering student at UBC with an option in biomedical engineering. I love working at the intersection of hardware and software — designing custom PCBs, writing embedded firmware, and building circuits that do something real.

I've built everything from wearable heart rate monitors with analog signal chains to FPGA-based games and 8051 assembly firmware. I'm as comfortable in the lab with an oscilloscope as I am writing C for a microcontroller at midnight.

Outside of engineering I'm into drumming, ultimate frisbee, and badminton. Always happy to chat about circuits, projects, or anything in between.

Embedded Systems · Biomedical

Heart Rate Monitor

Wearable biometric monitor built in embedded C on a PIC32 MCU. Captures heart activity with an IR LED–phototransistor pair, cascading LM358 op-amps, and a Schmitt Trigger. Real-time BPM graphed over UART in a Python GUI.

CPIC32UARTOp-AmpPython

Embedded Systems · Robotics

Magnetic Field Following Robot

Robot that follows a magnetic field using an STM32 MCU with an IR receiver and decoder. Measures pulse widths to capture custom data packets for wireless remote-to-robot communication. Validated with oscilloscopes and voltmeters.

CSTM32IRPWM

Firmware · Assembly

Reflow Oven Controller

Firmware written in 8051 Assembly for a CV-8052 soft processor. Interfaces with a Solid State Relay to control oven heating via interrupt-based PWM, maintaining precise temperatures across reflow profile states.

8051 ASMCV-8052PWMSSR

Tron Light Cycle Game

Interactive Tron-inspired game on a DE10-Lite FPGA. Implemented in C with pushbutton controls, switch-selectable speed settings, and real-time score display on HEX outputs. Full game logic in hardware-accelerated digital logic.
